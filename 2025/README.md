# Advent of Code 2025 in Gleam

## Setup & Running

1. Add your input data:

Place an `input.txt` file in the folder of the day you want to solve. 

2. Run the solutions:

  Since I wanted to keep the same structure as past years, I've put everything together in a way that doesn't quite fit with how Gleam works (AFAICT). Still, to run it one day, all you have to do is run:
  
  ```bash
  gleam run -m day_01/main
  ```
